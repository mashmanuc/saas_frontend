## Архітектурні рішення

### i18n Setup
- Використано `vue-i18n` з Composition API (`legacy: false`)
- Lazy loading перекладів через динамічний `import()`
- Підтримка форматування дат та чисел для кожної локалі
- Збереження вибраної мови в `localStorage`
- Синхронізація з бекендом через API

### Onboarding Flow
- Окремі потоки для студентів та тьюторів
- Покрокова навігація з можливістю пропуску
- Збереження прогресу на бекенді
- Можливість відкласти онбординг

### Checklist System
- Групування завдань за категоріями
- Система балів за виконання
- Відстеження прогресу в реальному часі
- Рекомендації наступного кроку
