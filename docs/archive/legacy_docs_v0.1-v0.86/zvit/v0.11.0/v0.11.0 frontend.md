# Звіт по фронтенду v0.11.0

## Огляд релізу
Версія **v0.11.0** сфокусована на завершенні усіх realtime-функцій та покращенні зручності роботи тьютора: чат, дошка, календар, профіль і i18n. Усі задачі з технічного завдання виконані, юніт-тести та Playwright-сценарії пройдені.

## Виконані блоки

### 1. Notifications
- Живі повідомлення з дропдауном, fallback-полінгом і маркуванням як прочитані.
- Відображення skeleton-ів, порожніх станів, кнопок “mark all”.

### 2. Realtime Chat (п.2 технічного завдання)
- Skeleton loader для історії, авто-скролл, статуси `sending/delivered/error`, retry + rate-limit 300 мс.
- Оптимістичне надсилання з clientId, підписка на канал `chat:user:{id}`, фільтрація за lessonId.
- Авто-ресинк історії після реконекту, notification pill для offline-staтусу.
- Додані юніт-тести Pinia store та Playwright e2e-сценарій (два користувачі).

### 3. Whiteboard polish
- Live cursors, undo/redo, авто-збереження, індикатори активності.

### 4. Calendar & Lessons
- Drag-resize подій, timezone-aware відображення, швидкі фільтри за статусами.
- Поведінка “select and edit” синхронізована між студентами та тьюторами.

### 5. Profile
- Autosave форми, завантаження аватара, валідації та підказки.

### 6. i18n
- Перемикання EN/UK, fallback та audit ключів (особливо для нових чат-рядків).

### 7. Глобальні покращення
- Error boundary + toast система, єдині утиліти для `notifyError/Info`, централізовані realtime store-и.

## Покриття тестами
- **Vitest**: усі модулі (chatStore, notificationsStore, dashboard, profile, lessons, relations) + coverage по профілю.
- **Playwright**: e2e для чату; інші smoke-сценарії лишились доступними як окремий проект.
- Кожний тестовий ран наразі зелений; повідомлення “ECONNREFUSED” у логах — очікувані емульовані офлайн-кейси.

## Пропозиції на v0.12 (frontend)
1. **Realtime Chat v0.12+**
   - Threaded/quoted відповіді, pinned messages, пошук по історії.
   - Push notifications у браузері при офлайні.
2. **Notifications 2.0**
   - Групування за типами, “Do Not Disturb” вікна, архів прочитаних.
3. **Whiteboard**
   - Експорт у PDF/PNG, snap-to-grid, бібліотека фігур.
4. **Calendar**
   - Масові операції (bulk move/cancel), інтеграція з Google Calendar.
5. **Performance & DX**
   - Code-splitting для найбільших модулів (dashboard/chat).
   - Документація для e2e-тестів + GitHub Actions job (Playwright).
6. **Accessibility & UX**
   - Audit для keyboard navigation у dashboard.
   - Темна тема / high-contrast режим.

## Висновок
Версія v0.11.0 закрила всі задачі беклогу та стабілізувала realtime-функціонал. Рекомендовано перейти до підготовки v0.12 із акцентом на advanced-можливостях чату, інтеграціях і продуктивності.
